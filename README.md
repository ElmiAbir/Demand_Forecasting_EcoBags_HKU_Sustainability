# Demand Forecasting & Inventory Optimization for Eco Bags

**HKU Sustainability Club Project**  
Year 2 IELM | The University of Hong Kong

### Project Objective
Built an end-to-end demand forecasting and inventory optimization system to support the HKU Sustainability Club’s material distribution.  
Predicted weekly demand for Eco Bags and recommended smart inventory policies to minimize costs while avoiding stockouts.

### Key Results
- Forecasted demand using Prophet model (capturing trend and seasonality)
- Calculated **Economic Order Quantity (EOQ)** = **512 units**
- Recommended **Reorder Point** = **23.9 units**
- Suggested **Safety Stock** = **42.2 units**
- Target stock level ≈ **66 units**
